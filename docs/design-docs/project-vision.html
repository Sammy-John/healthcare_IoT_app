Vision Statement

ConnectedCare helps people stay independent longer. With a voice-first, always-on health companion that keeps individuals, families, and healthcare professionals connected in real time. Simple, dignified, and designed for everyday life.

Problem Concept

  • Increasing demand for remote healthcare monitoring
  • Importance of interaction design in healthcare solutions
  • Focus on accessibility and user-centered design

The App

  • Utilizes Raspberry Pi to collect data from IoT health devices
  • Voice-controlled interface compatible with smart TVs
  • Access also available via smartphone

Target Audience

  • Users with disabilities, poor health, or chronic conditions
  • Caregivers/family members needing remote monitoring tools
  • Healthcare professionals using data insights to advise adjustments
