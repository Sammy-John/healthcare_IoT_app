Empowering Independant Living

What is ConnectedCare?

ConnectedCare is a smart-health-monitoring platform designed to support safer, more independant living for older adults and individuals with health challenges.

Utilising a micro-server,it gathers real-time data from IoT devices and presents it through a thoughtfully designed interface focused on accessibility.

Using voice-controlled navigiation, to intuitive visuals. Dive into our analysis, explore the design, and access the process of shaping the future of connected care.

What is the Healthcare IoT App?

This application is a smart health-monitoring portal that collects and displays real-time data from IoT devices using a Raspberry Pi server as its core.

Key Features

Voice-Activated Interface

Interact hands-free with your health data using simple voice commands.

Real-Time Monitoring

Live updates from wearables and medical IoT devices to ensure peace of mind.

Caregiver Alerts

Emergency notifications and status reports sent directly to caregivers.

Secure & Private

Data is encrypted and stored securely with user control over sharing settings.
